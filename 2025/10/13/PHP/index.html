PHP
Oct 13, 2025

一、核心特性

  • 跨平台性：支持Windows、Linux、macOS等操作系统，兼容Apache、Nginx等服务器。
  • 脚本语言：无需编译，直接由服务器解释执行。
  • 弱类型语言：变量类型无需显式声明，自动转换。
  • 面向对象与过程编程：同时支持两种编程范式，适合不同规模项目。
  • 丰富的扩展库：内置数据库操作（MySQL、PostgreSQL）、文件处理、网络通信等功能。

二、基础语法

  1. 标记
  2. 注释
    • // 这是 PHP 单行注释
    • /这是PHP多行注释/
  3. 变量
    • PHP变量
      • 变量以 $ 符号开始，后面跟着变量的名称
      • 变量名必须以字母或者下划线字符开始
      • 变量名只能包含字母、数字以及下划线（A-z、0-9 和 _ ）
      • 变量名不能包含空格
      • 变量名是区分大小写的（$y 和 $Y 是两个不同的变量）
    • global 关键字用于函数内访问全局变量
    • static 是函数的局部变量。每次调用该函数时，该变量将会保留着函数前一次被调用时的值。
  4. echo 和 print 语句
    • echo : 可以输出一个或多个字符串
    • print : 只允许输出一个字符串，返回值总为 1
  5. EOF(heredoc) 使用说明
    • 定义一个字符串的方法
  6. PHP 数据类型
    • String（字符串）
    • Integer（整型）
    • Float（浮点型）
    • Boolean（布尔型）
    • Array（数组）
    • Object（对象）
    • NULL（空值）
    • Resource（资源类型）
  7. PHP 类型比较
    • 松散比较：使用两个等号 == 比较，只比较值，不比较类型。
    • 严格比较：用三个等号 === 比较，除了比较值，也比较类型。
    • 例如，”42” 是一个字符串而 42 是一个整数。FALSE 是一个布尔值而 “FALSE” 是一个字符串。
  8. PHP 常量
    • PHP 中的常量是指一旦定义后其值不能被改变的标识符。
    • 常量值被定义后，在脚本的其他任何地方都不能被改变。
    • 常量可以用 define() 函数或 const 关键字来定义。
      • define():define(“GREETING”, “欢迎访问 Runoob.com”)
      • const CONSTANT_NAME = “value”
  9. PHP 字符串变量
    • 并置运算符 . 用于把两个字符串值连接起来
    • strlen() 函数返回字符串的长度（字节数)
    • strpos() 函数用于在字符串内查找一个字符或一段指定的文本.例如：strpos(“Hello world!”,”world”)
    • PHP String 参考手册：https://www.runoob.com/php/php-ref-string.html
  10. PHP 运算符
    • 在 PHP 中，赋值运算符 = 用于给变量赋值。算术运算符 + 用于把值加在一起。
    • 组合比较符：$c = $a <=> $b
      • 如果 $a > $b, 则 $c 的值为 1。
      • 如果 $a == $b, 则 $c 的值为 0。
      • 如果 $a < $b, 则 $c 的值为 -1。
